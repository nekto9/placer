### Фронт

Фронт на основе GravityUI и React hook form.

Дев-режим

```
yarn serve
```

Генерация хуков для rtk query (бэк должен быть запущен, т.к. json берется оттуда)

```
yarn apigen
```

Для фронта нужны файлы **.env.dev** и **.env.prod**. Данные из них используются только webpack, поэтому они нужны только локально.

Или нужно поменять логику в _scripts\buildPlugins.ts_ для плагина Dotenv.

#### Прочее

Тесты написаны только для компонента FileUpload, там же прикуручен воркер на ресайз изображений.

```
yarn test
```
